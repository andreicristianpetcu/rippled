Detailed Description

An immutable linear range of bytes.

A fully constructed Slice is guaranteed to be in a valid state. Default construction, construction from nullptr, and zero-byte ranges are disallowed. A Slice is lightweight and copyable, it retains no ownership of the underlying memory.
